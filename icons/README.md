# Ad Click Guard Icons

This directory should contain the following icon files for the Chrome extension:

- `icon16.png` - 16x16 pixel icon for browser toolbar
- `icon48.png` - 48x48 pixel icon for extension management
- `icon128.png` - 128x128 pixel icon for the Chrome Web Store

## Icon Design Suggestions

The icons should represent the extension's purpose of protecting users from accidental ad clicks. Consider designs that include:

- A shield symbol
- A stop sign
- A barrier or wall
- Red or warning colors to indicate protection

## Requirements

- Icons should be in PNG format
- Each icon should be sharp and clear at its specified size
- The design should be simple enough to be recognizable at small sizes
